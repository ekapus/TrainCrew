Air Brakes

Perform These Tests When Appropriate

  • Charge Train - To be performed when you couple up to cars that need to be moved.
  • Set and Release Test - To be performed when you need to ensure that the train's brakes are functioning correctly.
  • Continuity Test - To be performed if the trainline has been uncoupled and recoupled but no other changes have been made. Such as breaking a train over a crossing or changing out motive power.
  • Intermediate Air Test - To be performed enroute when cars are added or removed from the trainline. Only new cars need be tested.
  • Initial Air Test - To be performed when any crew takes control of a new train or if the trainline has been significantly modified.
